# FoundHer Website SEO Checklist

A comprehensive SEO implementation guide for the FoundHer website - empowering African women founders through strategic optimization.

## **Technical SEO**

### Meta Tags & HTML Structure
- [ ] Add proper `<title>` tags for each page/section
- [ ] Implement meta descriptions (150-160 characters)
- [ ] Add Open Graph tags for social media sharing
- [ ] Include Twitter Card meta tags
- [ ] Add canonical URLs
- [ ] Implement proper heading hierarchy (H1, H2, H3, etc.)
- [ ] Add alt text to all images
- [ ] Create XML sitemap
- [ ] Add robots.txt file

### Performance & Core Web Vitals
- [ ] Optimize images (WebP format, proper sizing)
- [ ] Implement lazy loading for images
- [ ] Minimize CSS and JavaScript bundles
- [ ] Enable GZIP compression
- [ ] Set up CDN for static assets
- [ ] Optimize Largest Contentful Paint (LCP)
- [ ] Improve First Input Delay (FID)
- [ ] Reduce Cumulative Layout Shift (CLS)

### Structured Data (Schema.org)
- [ ] Add Organization schema
- [ ] Implement Event schema for FoundHer events
- [ ] Add Person schema for leadership profiles
- [ ] Include WebSite schema with search action
- [ ] Add BreadcrumbList schema for navigation

## **Content SEO**

### Keyword Strategy
- [ ] Research primary keywords (African women entrepreneurs, women founders network, etc.)
- [ ] Identify long-tail keywords
- [ ] Map keywords to specific pages/sections
- [ ] Optimize for local SEO (country-specific terms)
- [ ] Research competitor keywords

### Content Optimization
- [ ] Optimize hero section copy for target keywords
- [ ] Write SEO-friendly section descriptions
- [ ] Add location-based content for 30+ countries
- [ ] Create unique descriptions for success stories
- [ ] Optimize CTA button text for conversions
- [ ] Add FAQ section addressing common queries

### Internal Linking
- [ ] Create logical internal link structure
- [ ] Add contextual links between sections
- [ ] Implement breadcrumb navigation
- [ ] Link to relevant external authority sites

## **Local & International SEO**
